# 鸽巢原理
## 定义
若k是一个正整数，将k+1个物体放入k个盒子中，则至少一个盒子包含两个或更多的物体。
## 推论
从一个具有 𝑘+1 个元素的集合到一个具有 𝑘 个元素的集合的函数 𝑓 不是一一对应的
## 广义鸽巢原理
如果将N个物体放进k个盒子当中，那么至少有一个盒子包含至少$\lceil \frac{N}{k} \rceil$个物体。

# 排列和组合
## 排列
### 定义
一组不同对象的排列是这些对象的有序序列。集合中 𝑟 个元素的有序序列称为 𝑟-排列.具有n个元素的集合的r-排列的数量记为P(n,r).
### 公式
*定理1*：:若𝑛 是正整数，且 𝑟 是满足 1≤𝑟≤𝑛 的整数，则对于一个有 𝑛 个不同元素的集合，其 𝑟-排列的数量为$$P(n, r) = n(n − 1)(n − 2) ··· (n − r + 1)$$
*推论1*：若n和r是满足$1 \leq r \leq n$的整数，那么$$P(n,r) = \frac{n!}{(n-r)!}$$
## 组合
### 定义
一个集合元素的 r-组合 是从集合中无序选择 r 个元素。因此，r-组合只是一个包含 r 个元素的子集。具有 n 个不同元素的集合的 r-组合的数量记作 C(n, r)或 $(_n^r)$ ，也称为二项式系数
### 公式
*定理2*：当$0 \leq r \leq n$时，具有n个元素的集合的r-组合数量为$$C(n,r) = \frac{n!}{(n-r)!r!}$$
*推论2*：设n和r是非负整数，且$r \leq n$，则$C(n,r) = C(n,n-r)$
## 组合证明
组合证明是指使用以下方法之一来进行恒等式的证明：
- 双计数证明：使用计数论证来证明恒等式两边以不同的方式计算相同的对象
- 双射证明：恒等式两边所计数的对象集合之间存在双射关系
# 二项式系数和恒等式
## 二项式定理
*定理*：设x和y是变量，n为非负整数，那么$$(x+y)^n = sum_{j=0}^n(_j^n)x^{n-j}y^j
 = (_0^n)x^n+(_1^n)x^{n-1}y+...+(_{n-1}^n)xy^{n-1}+(_n^n)y^n$$
 *换言之*，其实就是选取特定数量的项的意思~~，所以说知道加法原理乘法原理之后都不用管了~~
## 有用的恒等式
*定理1*：当$n \geq 0$，$$sum_{k=0}^{n}(_k^n) = 2^n$$
*换言之*，其实就是$(1+1)^n = 2^n$应用二项式定理得到的结果
*帕斯卡恒等式*：若n和k是整数，且满足$0 \leq k \leq n$，则C(n+1,k) = C(n,k-1) + C(n,k)
*换言之*，这个是杨辉三角的经典小推论

## 带重复的排列
*定理1*：允许重复的情况下，从n个对象的集合中选取r个对象的排列数为$n^r$
*换言之*，乘法原理，不用管
## 带重复的组合
*定理2*：当允许元素重复时，从一个包含n类元素的集合中选择r个元素的组合数是C(n+r-1,r) = C(n+r-1,n-1)
*换言之*，这个方法的意味是指**顺序不重要**下**可重复**选取物品的公式是**物品类型+选取个数-1**来C**选取个数**个

| 类型   | 是否允许重复 | 公式         |
| ---- | ------ | ---------- |
| r-排列 | 否      | P(n,r)     |
| r-组合 | 否      | C(n,r)     |
| r-排列 | 是      | n^r        |
| r-组合 | 是      | C(n+r-1,n) |
## 带有不可区分对象的排列
*定理3*：有n个对象，其中有$n_1$个不可区分的类型1对象，...，$n_k$个不可区分的类型k对象，不同排列的数量是$$\frac{n!}{n_1!n_2!...n_k!}$$
*换言之*，类似定序，即全排列之后进行序数的“剥离”，把不同的有序数要求的要求给削弱掉
## 将对象分配到盒子里
许多技术问题可以通过等效为将物体放入盒子里面来解决
- 物体可能相同/不同
- 盒子可能相同/不同
### 可区分物体，可区分盒子
将n个可区分物体分配到k个可区分盒子中的方法数为$\frac{n!}{n_1!n_2!...n_k!}$
### 不可区分物体，可区分盒子
将n个不可区分物体放入r个可区分盒子中的方法数为C(n+r-1,n)
### 可区分物体，不可区分盒子
对于将n个可区分物体放入j个不可区分盒子中的方式，目前并不存在简单的封闭公式
### 不可区分物体，不可区分盒子
同样不存在简单的封闭公式

| 物体\盒子 | 盒子可区分 | 盒子不可区分 |
|-----------|------------|--------------|
| 物体可区分 | $\displaystyle\frac{n!}{n_1!\,n_2!\cdots n_k!}$ | 无简单封闭公式（Bell 数求和） |
| 物体不可区分 | $C(n+r-1,\,n)$ | 无简单封闭公式（分拆数） |
## 母函数
母函数是一种使用多项式函数来对情况进行分析的方法，其特点有：
- 使用代数运算取代逻辑推理
- 涵盖所有情况
**基本思想**：把离散数列和幂级数一一对应起来，把离散数列间的相互结合关系对应成为幂级数间的运算关系，最后由幂级数形式来确定离散数列的构造
### 定义
对于序列$a_0,a_1,...,$
函数$G(x) = a_0 + a_1x +a_2x^2 + ...$称为该序列的母函数。
## 整数的拆分
可以按照下列要求进行拆分：是否有序，是否允许重复
其实也是将实际问题转换为整数的拆分问题在进行解决的
*定理1*：整数拆分成不同整数的和的拆分数(不允许重复)等于拆分成奇数的拆分数(允许重复)。
*定理2*：N 拆分成其他数之和但重复数不超过2，其拆分数等于它拆分成不被3整除的数的和的拆分数。
*定理3*：N 被拆分成一些重复次数不超过 k 次的整数的和，其拆分数等于被拆分成不被 k+1 整除的数的和的拆分数。
*定理4*：对任意整数 N，它被无序拆分成2的幂次的和的拆分方式一定唯一。
*定理5*：将 N 允许重复地有序拆分成恰好 r 个部分的方案数为C(N-1,r-1)。
定理6 将N允许重复地有序拆分成由数字$𝑎_1, 𝑎_2, ⋯ 𝑎_r$组成的方案数为下述公式的$𝑥_N$次方系数：$$1+(x^{a_1}+x^{a_2}+...+x^{a_r})+(x^{a_1})+x^{a_2}+...+x^{a_r})^2+(x^{a_1})+x^{a_2}+...+x^{a_r})^3...$$

### 定义

函数$G(x) = a_0 + \frac{a_1}{1!}x +\frac{a_2}{2!}x^2 + ...$称为该序列的指数型母函数。
